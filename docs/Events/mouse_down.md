# mouse_down

Emitted when any mouse button is pressed.

## Parameters

1. `mouse_down` : string - Event name.
2. `button` : number - ID of the mouse button.
3. `x` : number - X coordinate of the mouse.
4. `y` : number - Y coordinate of the mouse.

## Button IDs

| ID | Button   |
|----|----------|
| 1  | left     |
| 2  | right    |
| 3  | middle   |
| 4  | button 4 |
| 5  | button 5 |
