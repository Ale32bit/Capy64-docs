# key_up

Emitted when any keyboard key is released.

## Parameters

1. `key_up` : string - Event name.
2. `keyCode` : number - ID of the key.
3. `keyName` : string - Name of the key.
4. `modifiers` : integer - Flags of pressed modifiers.

## Modifiers flag

| Value | Key           |
|-------|---------------|
| 0     | None          |
| 1     | Left Shift    |
| 2     | Right Shift   |
| 4     | Left Alt      |
| 8     | Right Alt     |
| 16    | Left Control  |
| 32    | Right Control |
